# 🌦 Weather Forecast App

A simple Android Weather Forecast application built using Kotlin and Jetpack Compose.

## Features
- 🔍 Search weather by city
- 🌡 Live temperature
- 💧 Humidity
- ☁ Weather description
- 📍 City name
- ⚡ Real-time weather using OpenWeather API

## Tech Stack
- Kotlin
- Jetpack Compose
- Retrofit
- Gson Converter
- OpenWeather API
- Material 3
